# Registration Form

## Project Overview
This project is a simple Registration Form developed using HTML, CSS, and JavaScript. It provides a clean and user-friendly interface for users to enter their registration details. JavaScript is used to validate the input fields before allowing successful registration.

## Features
- User-friendly registration form
- Full Name input
- Email validation
- Phone Number validation
- Password and Confirm Password fields
- Checks for empty fields
- Password matching validation
- Success and error messages
- Responsive and modern design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```
Registration-Form/
│── index.html
│── style.css
│── script.js
└── README.md
```

## How to Run the Project
1. Download or clone the project.
2. Place all files in the same folder.
3. Open `index.html` in any web browser.
4. Fill in the registration details.
5. Click the **Register** button.

## Validation Performed
- All fields are required.
- Phone number must contain exactly 10 digits.
- Password and Confirm Password must match.
- Displays appropriate success or error messages.
